# Aamchi Budget — Mumbai Doodle Budget Tracker

This is a redesigned Mumbai Girl Budget Tracker inspired by a hand-drawn Mumbai collage/comic poster vibe.

## Default salary
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Other fixed income: editable
- Fixed costs: editable

## Features
- Daily log for expenses, income and savings
- Office breakfast tracker
- Rent + transport + electricity board
- Zomato / Zepto / Blinkit watchlist
- Surviving Thane score
- Mom contribution tracker
- First salary goals
- Travel fund, gifts for parents, Meta glasses fund, emergency fund
- Editable salary and fixed costs
- Export/import backup
- Local browser storage only

## Design
- Hand-drawn Mumbai doodle vibe
- Comic-style cards
- Thick black outlines
- Off-white paper background
- Local train, taxi, bus, chai and office-life collage elements

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
