# Comfort Vibe Tracker

A colorful daily money tracker with four selectable vibes:

1. Candy Pop Budget Tracker
2. Mumbai Girl Budget Tracker
3. Finance Tamagotchi
4. Sticker Scrapbook Tracker

## Features

- Daily income and expense log
- Candy Pop / Mumbai Girl / Finance Pet / Scrapbook theme switcher
- Money Mood meter
- Finance pet with level and XP
- Daily streak
- Mumbai-specific categories like office breakfast, Zomato, Zepto, Blinkit, mom contribution, rent, electricity, WiFi, transport
- Weekly scrapbook stickers
- Savings jars for Emergency Fund, Travel Fund, Gifts for Parents, and Meta Glasses
- Smart insights and watchlist
- Backup export/import
- Local browser storage only

## Vercel settings

- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public

## Privacy

All data is saved in browser localStorage. No bank login and no server upload.
