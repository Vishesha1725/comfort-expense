# Money Pet Room

A cute finance tracker where a dog pet lives inside a pastel room.

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Editable in Settings

## Features
- Cute dog finance pet (Mochi)
- Room gets prettier when you save money
- Room gets messy when you overspend
- Pet wakes up and moves around when you log daily
- Pet sleeps when you skip logging
- Savings target unlocks extra room items
- Moving coins near savings jar
- Calendar page flip animation
- Lamp brightness changes with room mood
- Plant grows with savings
- Daily log, savings goals, insights, pet care, history
- Export/import backup
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
