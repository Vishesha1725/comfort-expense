# Mumbai Girl Budget Tracker

A personalized, desi-modern daily budget tracker for Thane/Mumbai life.

## Default salary
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Both are editable inside the app.

## Main features
- Daily expense, income and saving log
- Editable income setup
- Editable fixed-cost setup
- Rent + transport + electricity dashboard
- Office breakfast tracker
- Zomato / Zepto / Blinkit watchlist
- Surviving Thane score
- Mom contribution tracker
- First salary goals
- Travel, gifts, Meta glasses and emergency fund
- Export/import backup
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
