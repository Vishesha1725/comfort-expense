# Comfort Daily Log

A colorful static personal finance website focused on a daily log instead of bank-statement uploads.

## Features
- Daily income and expense logging
- Monthly dashboard
- Colorful charts and animated UI
- Comfort score
- Budget health and insights
- History filters and search
- Export/import backup JSON
- Local browser storage only

## Deploy on Vercel
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public

## Structure
- public/index.html
- public/styles.css
- public/app.js
- vercel.json

## Privacy
All data stays in the browser using localStorage.
