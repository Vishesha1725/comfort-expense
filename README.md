# Finance Garden

A realistic pastel finance tracker where your money habits grow a garden.

## Style
- Realistic soft-pastel garden feel
- Not cartoonish
- Gentle depth, gradients, shadowing, and moving nature elements

## Features
- Savings plant flowers
- Emergency fund grows a big tree
- Travel fund grows a suitcase-shaped plant
- Overspending creates weeds
- Food delivery overspending creates snack monsters
- Daily logging creates watering rain and ripples
- Moving butterflies and drifting clouds
- Coin sunlight animation when you save
- Editable salary and fixed costs
- Daily log, savings goals, insights, history
- Export/import backup
- Local browser storage only

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
