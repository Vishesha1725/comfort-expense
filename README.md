# Money Quest — 3D Game-Style Money Tracker

This is Version 1 of the detailed themed trackers.

## Theme
3D Game-Style Money Tracker

## Default salary setup
- Gross salary: ₹60,000
- Deduction: 10%
- In-hand salary: ₹54,000

## Features
- Daily expense, income, and saving log
- XP system
- 10 levels from Money Baby to Finance Queen
- Badges
- Weekly quests
- Spend monsters
- Survival score
- 7-day spending map
- Budget power bars
- Quick add buttons for Zomato, Zepto, Blinkit, transport, snacks, coffee
- Export/import save file
- Browser localStorage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
