# Money Quest Pet — Ultimate

Final merged version of 3D Game-Style Money Tracker + Finance Tamagotchi.

## Character lineup
Queen character removed. Panda included.

Characters:
- Mochi — Money Pup
- Penny — Coin Fairy
- Panda Pockets — Budget Panda
- Snack Monster
- Shopping Goblin
- Cab Beast
- Bill Dragon
- Budget Owl
- Streak Squirrel

## Features
- 3D-style animated game home
- Cute finance pet with mood changes
- XP system and levels
- Level titles from Money Baby to Finance Legend
- Daily logging gives XP
- Savings give bonus XP and unlock accessories
- Pet sleeps if you skip logging
- Pet gets happy/excited when you save or hit target
- Pet gets stressed if overspending is high
- Spend monster arena
- Quest board with daily and weekly missions
- Reward chest with badges and character unlocks
- Savings goals with progress bars
- Stats dashboard
- Daily log and searchable history
- Editable salary and fixed costs
- Export/import backup
- Local browser storage only

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
