# Money Mood Island

A cute animated budget tracker where your money habits become an island.

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Editable in Settings

## Features
- Animated island home
- Weather system: sunny, cloudy, rainy, storm, sparkly
- Moving waves, floating clouds, rocking boat, swaying palm trees
- Savings create flowers, treasure and sunshine
- Overspending creates storm weather
- Daily logging keeps the island calm
- Rent affects the house
- Transport activates island routes
- Food app spends create cafe stalls
- Mom contribution lights a heart lighthouse
- WiFi/electricity power the signal tower
- Savings Cove with goals
- Weekly Mood Map
- Island Insights
- Daily Log Beach Board
- Searchable history
- Export/import backup
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
