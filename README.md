# Sticker Scrapbook Tracker

A creative, scrapbook-style daily budget tracker.

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Everything is editable.

## Features
- Add each expense as a sticker
- Mood sticker for each spend
- Regret / Worth it / Necessary / Smart labels
- Weekly scrapbook summary
- Colorful category tags
- Savings jar visual
- Goal tracking
- Daily journaling notes
- History search and filters
- Export / import backup
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
