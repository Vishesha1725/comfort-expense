# Doodle Money Bakery

A doodle-style money tracker with playful bakery visuals, moving objects, and a fun handwritten look.

## Features
- Doodle bakery theme with fun font
- 3D-ish pastry display case
- Receipt printer animation when you add expenses
- Oven glow when income/activity is present
- Coffee cup with moving steam
- Savings jar with floating coins
- Celebration cake that grows tier-by-tier as savings progress increases
- Sold Out sign when the budget goes negative or gets stressed
- Daily log, savings, insights, settings, history
- Export/import backup
- Vercel-ready static files

## Default setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
