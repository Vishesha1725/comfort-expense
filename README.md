# Finance Tamagotchi

An advanced, playful, pet-based money tracker.

## Default salary setup
- Gross salary: ₹66,000
- In-hand salary: ₹60,000
- Everything is editable in the app.

## Features
- Cute animated money pet
- Pet mood changes based on spending and saving behavior
- Daily log for expense, income, saving, and breakfast/snack entries
- Pet leveling system with XP
- Logging streaks
- Savings goals
- Accessories unlocked from savings progress
- Daily and weekly challenges
- Achievements
- Smart pet insights
- History filters
- Export/import backup
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
