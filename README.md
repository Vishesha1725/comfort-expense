# Money Quest Pet — Pixel RPG Edition

A sharp pixel-style finance tracker with a proper safe-spend engine, RPG missions, spend monsters, inventory, and local-only storage.

## Key changes included
- Pixel RPG theme, sharper and less soft
- No queen character
- Panda Pockets included
- Mom contribution is not treated as fixed expenditure by default
- Fixed protected costs default to:
  - Rent: ₹23,000
  - Furniture rent: ₹2,000
  - WiFi: ₹499 only from September 2026 by default
- All numbers are editable
- WiFi start month is editable
- Mom bucket has optional protect toggle, default off
- Category budgets are editable

## Advanced features
- Advanced Daily Safe Spend engine
- Pixel sprite characters, not emoji UI
- Risk Arena with monster stages: Dormant, Growing, Angry, Boss Mode
- Daily missions and weekly challenges
- Vault goals and Family Bucket
- Finance Terminal with projected end balance
- Month-end boss logic
- Inventory with characters, badges, and items
- Export CSV
- Download monthly summary
- Export/import JSON backup
- Reset current month
- Reset all data
- Local browser storage only

## Vercel settings
- Framework Preset: Other
- Build Command: leave empty
- Install Command: leave empty
- Output Directory: public
